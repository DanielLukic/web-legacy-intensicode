Info

Intelli/J IDEA plugin providing generic simple syntax highlighting.

The included configuration gives you (still incomplete) highlighting for Ruby. You can easily extend this configuration or add support for other languages by editing and/or cloning simple configuration files.

Installation

Use the IDEA plugin manager.

Open Issues

These are just the more general issues with the current plugin version:

  • Lexer doesn't cope well with some ways of changing source code. Strange..
  • Highlighting a longer piece of source code can be pretty slow. Especially with lots of external (ruby/groovy) rules..
  • Arranging syntax rules in the right order can be annoying. A simple lexer-like configuration would be cool..
  • This projects needs a real home. With issue tracking etc..
